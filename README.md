Why choose the RTS framework?
- Free and for public use.
- Flexible and intuitive routing system.
- Support for the MVC pattern.
- A ready-made system of access rights for users and user groups.
- Extensible template system.
- Database structure organization from code.
- A system for checking and correcting differences in database structure.
- To create an application on the user side, SCSS and TypeScript are used.
- A flexible and extensible content editor has been implemented.
- Convenient, centralized work with sessions, cookies, and request parameters.
- Middleware support.
- Powerful debugging tool.
- Added helpers such as pagination, redirection, client responses, and others.
- Support for environment variables.
- A query builder for the MySQL DBMS has been implemented.
- You can add support for other DBMSs by implementing methods for your DBMS.
- Ready-made assembler for npm.
- Working with the main components of the system via the command line or web interface.
